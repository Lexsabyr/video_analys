# AI Video Moderation with Gemini

AI система модерации видео на Python с использованием Gemini API.

## Возможности

### Анализ кадров видео
Система автоматически:
- извлекает кадры из видео
- анализирует изображения через Gemini
- ищет:
  - оружие
  - насилие
  - кровь
  - наркотики
  - алкоголь
  - курение
  - экстремизм
  - свастику
  - сексуальный контент
  - пожары
  - взрывы
  - текст на изображении (OCR)

---

### Анализ аудио
Система:
- извлекает аудио из видео
- преобразует его в mp3
- отправляет в Gemini
- делает транскрипцию речи
- ищет:
  - угрозы
  - агрессию
  - насилие
  - экстремизм
  - мат
  - bullying
  - hate speech
  - наркотики
  - алкоголь
  - оружие

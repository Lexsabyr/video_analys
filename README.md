# AI Video Moderation with Gemin

AI система модерации видео на Python с использованием Gemini API.

## Возможности

### Анализ кадров видео
Система автоматически:
- извлекает кадры из видео
- анализирует изображения через Gemini
- ищет:
  - оружие
  - насилие
  - кровь
  - наркотики
  - алкоголь
  - курение
  - экстремизм
  - свастику
  - сексуальный контент
  - пожары
  - взрывы
  - текст на изображении (OCR)

---

### Анализ аудио
Система:
- извлекает аудио из видео
- преобразует его в mp3
- отправляет в Gemini
- делает транскрипцию речи
- ищет:
  - угрозы
  - агрессию
  - насилие
  - экстремизм
  - мат
  - bullying
  - hate speech
  - наркотики
  - алкоголь
  - оружие

---

### Генерация отчета
После анализа создается:
report.txt

# Получение Gemini API Key

## 1. Перейти в Google AI Studio

Open:

https://aistudio.google.com/

---

## 2. Войти в Google Account

Нужен обычный Google аккаунт.

---

## 3. Создать API Key

1. Нажать:

```text
Get API key
```

2. Потом:

```text
Create API key
```

3. Скопировать созданный ключ.

Пример:

```text
AIzaSyXXXXXXXXXXXX
```

---

## 4. Вставить ключ в код

Открыть файл:

```text
main.py
```

Найти строку:

```python
API_KEY = "YOUR_API_KEY"
```

И заменить на свой ключ:

```python
API_KEY = "AIzaSyXXXXXXXXXXXX"
```

---

# Лимиты бесплатного Gemini API

Free Tier имеет ограничения.

Возможны ошибки:

```text
429 RESOURCE_EXHAUSTED
```

или:

```text
503 UNAVAILABLE
```

Это нормально.

Причины:
- превышен лимит запросов
- сервер Gemini перегружен

В коде реализованы:
- автоматическое ожидание
- повторная отправка запросов
